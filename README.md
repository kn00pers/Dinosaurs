Dinosaurs App is a SwiftUI-based mobile application for dinosaur enthusiasts. It allows users to explore detailed information about predatory dinosaurs, their appearances in movies, and their geographical locations. With an intuitive interface, search functionality, and map integration, the app offers an engaging experience for all dino lovers!

---

## Features ğŸš€

- **Dinosaur List**: View a catalog of predatory dinosaurs with images, names, and types.  
- **Search and Filter**: Quickly find dinosaurs using a search bar or filter them by type.  
- **Sorting**: Toggle alphabetical sorting of the dinosaur list with a single tap.  
- **Detailed View**: Explore detailed information about each dinosaur, including:  
  - Appearance in movies  
  - Memorable movie scenes  
  - Links for further reading  
  - Current geographical locations displayed on a map  
- **Interactive Map**:  
  - See the locations of all apex predators.  
  - Switch between standard and satellite map styles.  
- **Dark Mode Support**: A sleek, dark theme for an immersive experience.
